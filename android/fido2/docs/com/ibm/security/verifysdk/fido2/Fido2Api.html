      • buildAuthenticatorAttestationResponse

         final AuthenticatorAttestationResponse buildAuthenticatorAttestationResponse(FragmentActivity activity, Executor executor, BiometricPrompt.PromptInfo.Builder promptInfoBuilder, String aaGuid, String keyName, Byte flags, PublicKeyCredentialCreationOptions options, String nickName)

        Builds an AuthenticatorAttestationResponse for the given parameters.

        This function constructs an AuthenticatorAttestationResponse, which represents the response from the authenticator during the attestation process. It is used to attest to the creation of a new public key credential. The response includes various data such as client data, attestation statement, and authenticator data.

        Parameters:
        activity - The FragmentActivity instance to display UI prompts.
        executor - The Executor instance to run asynchronous tasks.
        promptInfoBuilder - The PromptInfo.Builder instance to build the biometric authentication dialog.
        aaGuid - The AAGUID (Authenticator Attestation Globally Unique Identifier).
        keyName - The name of the key in the Android KeyStore.
        flags - The flags indicating the capabilities and status of the authenticator.
        options - The PublicKeyCredentialCreationOptions containing the parameters for creating the public key credential.
        nickName - The nickname associated with the authenticator.
        Returns:

        The constructed AuthenticatorAttestationResponse.

      • buildAuthenticatorAssertionResponse

         final AuthenticatorAssertionResponse buildAuthenticatorAssertionResponse(FragmentActivity activity, Executor executor, BiometricPrompt.PromptInfo.Builder promptInfoBuilder, String keyName, Byte flags, PublicKeyCredentialRequestOptions options, String message)

        Builds an AuthenticatorAssertionResponse for the given parameters.

        This function constructs an AuthenticatorAssertionResponse, which represents the response from the authenticator during the assertion process. It is used to attest to the authentication of an existing public key credential. The response includes various data such as client data, authenticator data, and signature generated by biometric authentication.

        Parameters:
        activity - The FragmentActivity instance to display UI prompts.
        executor - The Executor instance to run asynchronous tasks.
        promptInfoBuilder - The PromptInfo.Builder instance to build the biometric authentication dialog.
        keyName - The name of the key in the Android KeyStore.
        flags - The flags indicating the capabilities and status of the authenticator.
        options - The PublicKeyCredentialRequestOptions containing the parameters for the assertion process.
        message - Optional additional message to include in the assertion.
        Returns:

        The constructed AuthenticatorAssertionResponse.
