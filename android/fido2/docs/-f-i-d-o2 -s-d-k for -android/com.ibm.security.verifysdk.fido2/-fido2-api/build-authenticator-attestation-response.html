buildAuthenticatorAttestationResponse

suspend fun buildAuthenticatorAttestationResponse(activity: FragmentActivity, executor: Executor, promptInfoBuilder: BiometricPrompt.PromptInfo.Builder, aaGuid: String, keyName: String, flags: Byte, options: PublicKeyCredentialCreationOptions, nickName: String): AuthenticatorAttestationResponse

Builds an AuthenticatorAttestationResponse for the given parameters.

This function constructs an AuthenticatorAttestationResponse, which represents the response from the authenticator during the attestation process. It is used to attest to the creation of a new public key credential. The response includes various data such as client data, attestation statement, and authenticator data.

Return

The constructed AuthenticatorAttestationResponse.

Parameters

activity

The FragmentActivity instance to display UI prompts.

executor

The Executor instance to run asynchronous tasks.

promptInfoBuilder

The PromptInfo.Builder instance to build the biometric authentication dialog.

aaGuid

The AAGUID (Authenticator Attestation Globally Unique Identifier).

keyName

The name of the key in the Android KeyStore.

flags

The flags indicating the capabilities and status of the authenticator.

options

The PublicKeyCredentialCreationOptions containing the parameters for creating the public key credential.

nickName

The nickname associated with the authenticator.
