Class UpdateCredentialArgs

  • All Implemented Interfaces:
    @Serializable() 
public final class UpdateCredentialArgs

    Data class representing the arguments required to update a credential.

    The UpdateCredentialArgs class encapsulates all the necessary information for updating a credential, including its state, associated schema, definition, attributes, and other optional properties. This class is used to provide the required fields for modifying or creating a credential update request.
