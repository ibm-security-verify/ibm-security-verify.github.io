Package com.ibm.security.verifysdk.dc.api

Class VerificationsApi

  • All Implemented Interfaces:
    
public class VerificationsApi
extends BaseApi

    Provides an API interface for managing verifications API endpoints in IBM Verify Identity Digital Credentials.

    This class supports operations to retrieve, update, and delete verification records through the backend service. It communicates via HTTP and handles JSON (de)serialization using Kotlinx Serialization with experimental APIs.

    Example usage:

    val api = VerificationsApi(baseUrl)
val result = api.getAll(accessToken = "your-access-token")
    Since:

    3.0.7

         Result<List<VerificationInfo>> getAll(HttpClient httpClient, URL url, String accessToken, Map<String, String> additionalHeaders, VerificationState state)

        Retrieves a list of verifications from the API.

        This function sends a GET request to the specified URL or the default URL for retrieving a list of verifications. If the request is successful, it returns a list of VerificationInfo objects. In case of a failure, it returns a failure result with the associated error message or exception.

        Parameters:
        httpClient - The HTTP client used to perform the request.
        url - The URL to send the GET request to.
        accessToken - The access token used for authentication.
        additionalHeaders - Optional map of additional headers to include in the request.
        state - The verification state to filter the verifications by.
        Returns:

        A Result containing either a list of VerificationInfo objects if the request is successful, or a failure result with an error message if the request fails.
