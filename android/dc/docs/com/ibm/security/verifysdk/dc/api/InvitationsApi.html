Package com.ibm.security.verifysdk.dc.api

Class InvitationsApi

  • All Implemented Interfaces:
    
public class InvitationsApi
extends BaseApi

    Provides an API interface for managing and interacting with invitations API endpoints in IBM Verify Identity Digital Credentials.

    This class allows clients to retrieve, delete, and process invitation records. It communicates with a backend service via HTTP requests and handles response serialization.

    Example usage:

    val api = InvitationsApi(baseUrl)
val invitations = api.getAll(accessToken = "your-token")

    3.0.7

        This function sends a PUT request to the specified URL or the default URL for processing an invitation.
