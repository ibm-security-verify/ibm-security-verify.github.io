Class Format

  • All Implemented Interfaces:
    @Serializable() 
public final class Format

    Represents various credential formats supported in a digital credentialing system.

    This class holds different representations of credentials, such as JWT-based and Linked Data Proof (LDP) formats. It provides optional fields for different credential and presentation formats that may be used in verifiable credential exchanges.
