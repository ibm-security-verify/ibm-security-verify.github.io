Package com.ibm.security.verifysdk.dc.model

Class VerificationInfo

  • All Implemented Interfaces:
    @Serializable() 
public final class VerificationInfo

    Data class representing the information required for a verification process.

    This class contains details related to the verification process, including the role, state, associated credentials, proof request and response, connection details, and other relevant metadata.

    Since:
