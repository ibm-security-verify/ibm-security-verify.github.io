Class JsonLdCredential

  • All Implemented Interfaces:
    com.ibm.security.verifysdk.dc.model.CredentialDescriptor 
    @Serializable() 
public final class JsonLdCredential
 implements CredentialDescriptor

    Represents a JSON-LD formatted credential used in decentralized identity systems.

    This class contains details about a credential issued in JSON-LD format, including its metadata, issuer information, connection details, and properties.
